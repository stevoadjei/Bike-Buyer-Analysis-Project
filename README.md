# Bike-Buyer-Analysis-Project
# 1. Project Overview and Business Questions (The "Why")
# Project Goal
The primary objective of this project was to analyse the provided bike-buyer dataset to identify the key demographic and behavioural segments most likely to purchase a bicycle. This analysis aims to provide actionable insights into marketing and sales strategies.
# Core Business Questions
The analysis was structured to answer the following key questions:<br>
•	How does a potential buyer's age influence their decision to purchase a bike? <br>
•	Is there a correlation between income and the likelihood of purchasing a bike?<br>
•	Do different commute distances show varying rates of bike purchases?<br>
•	How do marital status and gender relate to the purchasing decision?<br>
